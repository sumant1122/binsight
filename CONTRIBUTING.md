# Contributing to Binsight

Thank you for your interest in Binsight! We welcome contributions from everyone.

## How to Contribute

### 🐛 Bug Reports
If you find a bug, please open an issue and include:
- Your operating system and version.
- The version of `binsight` you are using.
- A reproduction case (if possible).

### ✨ Feature Requests
Open an issue describing the feature you'd like to see and why it would be useful.

### 🛠 Pull Requests
1. Fork the repository.
2. Create a new branch for your feature or fix.
3. Ensure your code follows the existing style.
4. Run `cargo build` and ensure there are no warnings.
5. Submit a pull request!

## Development Setup
You will need the latest stable version of Rust installed.

```bash
git clone https://github.com/sumant1122/binsight.git
cd binsight
cargo build
```
